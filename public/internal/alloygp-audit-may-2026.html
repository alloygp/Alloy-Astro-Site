Security Headers

Vercel response headers were checked against OWASP recommendations. HSTS is correctly configured. Three additional headers are missing and should be added via vercel.json.

Header Current State Required Value Risk if Missing
Strict-Transport-Security Present max-age=63072000 (2yr) N/A — configured correctly
X-Frame-Options Missing SAMEORIGIN Clickjacking vulnerability
X-Content-Type-Options Missing nosniff MIME-type sniffing attacks
Referrer-Policy Missing strict-origin-when-cross-origin Referrer data leakage
Add to vercel.json
{
  "headers": [
    {
      "source": "/(.*)",
      "headers": [
        {
          "key": "X-Frame-Options",
          "value": "SAMEORIGIN"
        },
        {
          "key": "X-Content-Type-Options",
          "value": "nosniff"
        },
        {
          "key": "Referrer-Policy",
          "value": "strict-origin-when-cross-origin"
        }
      ]
    }
  ]
}
